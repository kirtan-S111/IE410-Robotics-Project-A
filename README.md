# IE410: Introduction to Robotics – Project Part A

This repository contains the implementation of the Winter 2026 robotics project focused on robotic object manipulation using Braccio robot arms.

---

## Project Objectives

- Understand the kinematic control of robotic manipulators with two-finger grippers
- Implement autonomous pick-and-place operations using predefined trajectories
- Integrate vision-based sensing for perception-assisted manipulation
- Develop coordination strategies for object transfer between two robotic arms
- Gain practical experience in robot motion planning, grasping, perception, and task sequencing

---

## Repository Structure

### 1. Blind Pick-and-Place Manipulation
**File:** `Task 1.ino`

- Pre-programmed robotic pick-and-place operation without visual feedback
- The robot transfers an object from Point A to Point B using predefined motions
- Demonstrates basic robot trajectory planning and gripper control

---

### 2. Vision-Based Pick-and-Place
**File:** `Task 2.m`

- Camera-assisted object detection and robotic manipulation using MATLAB
- Uses a webcam or smartphone camera for object localization
- Robot motion is adjusted dynamically based on visual input

---

### 3. Dual-Arm Object Handover
**File:** `Task 3.ino`

- Coordinated object transfer between two robotic arms
- Demonstrates synchronized grasping and release operations
- Highlights multi-robot coordination and bimanual manipulation
